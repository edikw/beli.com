
# beli.com

## Project setup
```
git clone https://github.com/edikw/beli.com.git
change directory to beli.com
npm install
```

### Compiles and hot-reloads for development
```
npm run serve
```

### Compiles and minifies for production
```
npm run build
```

### Run your tests
```
npm run test
```

### Lints and fixes files
```
npm run lint
```
